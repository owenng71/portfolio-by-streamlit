# My Streamlit Portfolio

### 1. Clone the repository
```
git clone <repo-link>
```

### 2. Navigate into the project folder
```
cd <your-folder>
```

### 3. Install the packages
In case you don't have Streamlit yet.
```
pip install streamlit
```

### 4. Run the project
In your terminal:
```
streamlit run main.py
```
